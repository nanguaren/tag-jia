## ✨ 核心功能

- **批量标签管理**
  - 一键添加多个标签
  - 正则表达式替换标签
- **智能建议**
  - 输入时自动补全高频标签

---

## 📥 安装方法

### 方式一：通过 Obsidian 安装
1. 打开 Obsidian → 设置 → 社区插件 → 浏览
2. 搜索 "TagJia" → 安装并启用
3. 重启 Obsidian
